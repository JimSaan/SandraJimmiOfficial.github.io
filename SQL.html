Comprehensive Data Cleaning and Standardization for Nashville Housing Dataset

This SQL project focuses on the cleaning, standardization, and transformation of the Nashville Housing dataset. Initially, the data is extracted and the sale dates are standardized for uniformity. The project addresses missing property addresses by leveraging other records with the same Parcel ID. Subsequently, the Property and Owner addresses are broken down into individual columns for Address, City, and State, allowing for more granular analysis. The "Sold as Vacant" field is converted from 'Y' and 'N' to 'Yes' and 'No', enhancing the readability and consistency of the data.

The project also takes an essential step of removing duplicate records based on a set of key columns, ensuring data integrity. Unused columns are dropped to optimize the database, and finally, examples of importing data using the OPENROWSET and BULK INSERT methods are provided. Through these steps, this project demonstrates a comprehensive approach to data preprocessing, enhancing the quality and usefulness of the Nashville Housing dataset for subsequent analytics tasks.

COVID-19 Data Analysis Using Advanced SQL Techniques

This project involves the in-depth exploration and analysis of COVID-19 data, leveraging advanced SQL techniques such as joins, common table expressions (CTEs), temporary tables, window functions, aggregate functions, and data type conversions. The analysis covers several important aspects related to the COVID-19 pandemic. This includes the calculation of death rates, infection rates in relation to a country's population, and the highest infection and death counts across different countries and continents. Moreover, the study delves into vaccination trends, calculating the percentage of population vaccinated in various regions. The creation of SQL views facilitates future data visualization and further analysis. The project showcases the power of SQL in extracting meaningful insights from complex, real-world datasets.

COVID-19 Analysis Queries for Tableau Visualization

These SQL queries are designed to fetch, organize and compute essential information from the 'CovidDeaths' and 'CovidVaccinations' tables. The information includes calculating total cases, total deaths, death percentage, total death count per location, highest infection count and the percentage of the population infected by COVID-19. It also includes detailed data for COVID-19 vaccinations. The resulting data is meant for analysis and visualization using Tableau.
